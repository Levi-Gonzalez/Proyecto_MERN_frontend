# Proyecto_MERN_frontend
Proyecto con stack MERN utilizando tecnologías como TaildwindCSS Headless UI, Context API, Socket IO Utilizando la arquitectura de MVC


"Desarrollé una aplicación Full Stack MERN que permite a los usuarios crear y gestionar notas y tareas.
Implementé funciones para agregar y eliminar usuarios con roles de colaboradores, que pueden marcar tareas como completadas. 
Utilicé MongoDB para almacenar los datos de manera segura mediante JWT (JSON Web Tokens) para la autenticación. 
La arquitectura MVC garantiza la organización y la escalabilidad, y Jest fue empleado para pruebas rigurosas. 
Logré un exitoso despliegue en Netlify y Railway,
demostrando habilidades sólidas en desarrollo Full Stack con enfoque en seguridad y funcionalidad."
